## 🛠️ Tech Stack
- **Framework:** Flutter (3.41.7)
- **Architecture:** Clean Architecture (Feature-based)
- **State Management:** Flutter Bloc (Cubit)
- **Navigation:** GoRouter
- **Dependency Injection:** GetIt
- **Networking:** Dio
- **Responsiveness:** Flutter ScreenUtil
- **Localization:** Flutter Gen-L10n
